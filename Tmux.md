# Tmux Cheat Sheet

Các phím tắt cơ bản để quản lý cửa sổ (window) và khung (pane) trong Tmux.

**Khởi động:**
tmux


**Phím lệnh gốc (Command Prefix):**
* Mặc định: `CTRL` + `B` (Gọi tắt là `Prefix`)

### Các lệnh điều khiển

| Hành động | Tổ hợp phím | Ghi chú |
| :--- | :--- | :--- |
| **Tạo cửa sổ mới** (New window) | `Prefix` + `c` | |
| **Chia màn hình ngang** (Split horizontal) | `Prefix` + `Shift` + `%` | Chia đôi màn hình thành trái/phải. |
| **Chia màn hình dọc** (Split vertical) | `Prefix` + `Shift` + `"` | Chia đôi màn hình thành trên/dưới. |
| **Di chuyển giữa các Pane** | `Prefix` + `Phím điều hướng` | (Trái/Phải/Lên/Xuống) |
